修改博客配置文件

  • 打开 _config.yml配置文件
  • 找到 theme:
  • 把 Hexo 默认的 lanscape 修改成 hexo-theme-next 。
  • 找到 # Site，添加博客名称，作者名字等。
  • 在 language 后面填入 en 或者 zh-CN，选择英文或者中文。
  • 找到 # URL, 填入 url。比如 url: https://happy4java.github.io
